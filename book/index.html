Crei Kernel

Crei Kernel

Crei is a simple kernel forged purely out of boredom. It is created as an educational tool to learn more about C and systems programming.

Compiling & Execution

Just run make and follow up with this command and you're good to go:

$ qemu-system-x86_64 -cdrom build/crei.iso -m 256M

Getting Started

There is no getting started. You're not going to daily drive Crei, are you? Just head to shell docs to learn about a few commands that Crei supports. Maybe run help in the console too for a quick summary of commands while you're at it.
